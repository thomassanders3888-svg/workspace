## Deployment Options

### Vercel (Recommended)
1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel --prod`

### Netlify
1. Drag `sites/birth-rate-viz/` folder to https://app.netlify.com/drop

### Cloudflare Pages
1. Connect Git repo or upload via CLI
2. Build command: none (static)
3. Output: `/`
