## Agent Architecture

### Sales Closer Agent
- Qualifies leads via WhatsApp
- Schedules discovery calls
- Sends proposals
- Follows up automatically

### Lead Researcher Agent
- Scrapes LinkedIn, industry directories
- Enriches contact data
- Segments by urgency/pain points
- Daily lead list to Sales agent

### Automation Builder Agent
- Creates custom OpenClaw skills
- Builds client-specific workflows
- Manages deployments
- Monitors system health

### Proposal Writer Agent
- Generates SOW documents
- Creates case studies
- Drafts email sequences
- Updates pricing dynamically

### Client Success Manager Agent
- Monitors client metrics
- Sends weekly reports
- Identifies churn risk
- Triggers upsell conversations
