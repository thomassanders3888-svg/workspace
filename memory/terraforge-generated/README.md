 

# TerraForge

TerraForge is a voxel-based multiplayer game that combines the simplicity of block building with strategic gameplay elements. It's designed to be accessible yet engaging, perfect for players of all ages and skill levels.

## Features

- **Block Building**: Construct intricate structures using an extensive variety of blocks.
- **Multiplayer Mode**: Join and create your own worlds with other players.
- **Creative & Survival Modes**: Choose between creative freedom or survival challenges.
- **Customizable Worlds**: Modify and explore the world you create.
- **Realistic Physics**: Experience gravity, fluid dynamics, and more in a realistic manner.

## Installation

TerraForge is available for Windows, macOS, and Linux. Follow these steps to install:

### Windows

1. Download the latest installer from our [GitHub releases page](https://github.com/TerraForgeTeam/TerraForge/releases).
2. Run the downloaded `.exe` file.
3. Follow the on-screen instructions to complete the installation.

### macOS

1. Download the latest `.dmg` file from our [GitHub releases page](https://github.com/TerraForgeTeam/TerraForge/releases).
2. Open the downloaded `.dmg` file.
3. Drag the TerraForge icon to your Applications folder.
4. Launch TerraForge from your Applications folder.

### Linux

1. Download the latest tarball from our [GitHub releases page](https://github.com/TerraForgeTeam/TerraForge/releases).
2. Extract the tarball using a terminal:
   ```sh
   tar -xvf terraforge-<version>.tar.gz
   ```
3. Navigate to the extracted directory.
4. Run TerraForge:
   ```sh
   ./terraforge
   ```

## Usage

Once installed, launching TerraForge is straightforward:

1. Open the TerraForge application.
2. Create a new world or join an existing one.
3. Choose Creative or Survival mode and begin building!

For more detailed instructions and tips, check out our [official documentation](https://docs.TerraForgeTeam.com/).
