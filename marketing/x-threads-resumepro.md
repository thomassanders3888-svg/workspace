**Tweet 2/7:**

- Semantic matching (knows "Python" = "Python 3")
- Context understanding (not just buzzwords)
- Skills taxonomy (grouped competencies)
- Density scoring (balance, not stuffing)

Old tricks don't work. Authentic matching does.

**Tweet 3/7:**
The 3-layer keyword strategy:

LAYER 1: Job title synonyms
"Project Manager" = "PM" = "Program Manager" = "Project Lead"

LAYER 2: Core skills as written
If the JD says "React.js," write "React.js" not "JavaScript framework"

LAYER 3: Soft skills with metrics
"Led team" ‚Üí "Led 5-person team" + result

---

**Tweet 4/7:**

‚úì Summary (2-3 core competencies)
‚úì Skills section (keyword list)
‚úì Experience bullets (context + results)
‚úì Education/certifications (relevant coursework)
