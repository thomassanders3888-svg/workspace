# GameHub Screenshot Specification

## Screenshot Purpose
Primary marketing asset showcasing GameHub's unified game library interface

## Visual Layout

### Main View: Dashboard
- **Header**: Clean navigation with user profile, search bar, notification bell
- **Hero Section**: Featured game banner (currently playing/installed)
- **Game Grid**: 3-column responsive layout showing game covers

### Games Visible (8-12 titles)
Primary row highlights:
| Position | Game | Visual Treatment |
|----------|------|------------------|
| 1 | Cyberpunk 2077 | Recently played, glowing accent |
| 2 | Elden Ring | 45h played badge |
| 3 | Baldur's Gate 3 | Update available indicator |
| 4 | Hades II | Installed badge |
| 5 | Steam logo row | Steam library integration visible |
| 6 | Epic Games tile | Epic library synced |
| 7 | GOG Galaxy | DRM-free games section |
| 8 | Xbox Game Pass | Subscription games highlighted |

### UI Elements to Feature
- **Smart Filters**: "All Games" | "Installed" | "Recently Played" | "Backlog"
- **Playtime Stats**: Hours counter per game
- **Progress Tracking**: Small progress bars for achievements/completion
- **Sync Status**: Green checkmarks indicating library sync complete

### Color Palette
- Dark mode by default (#1a1a2e background)
- Accent: Electric blue (#4a90d9)
- Success green for sync indicators
- Game cover art provides color variety
